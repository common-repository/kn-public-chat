KN Public Chat is a free WordPress Plugin that lets your visitors and visitor from anyone who install this plugin can chat together in 1 public chat room.

== Description ==

* For anyone who has cbox.ws premium acc, you can use this plugin for your own purpose also!
* We use premium chatbox.ws as chatroom we will share the room! Anyone who install this plugin will get connected via this chat room. This plugin was intergrate in wordpress display name, so anyone who already registered in your wordpress, they can chat with their display name anytime.

= Feature =

* Implement your own cbox.ws premium account
* Widget
* Gravatar Intergration
* Display Name (from wordpress profile)
* Your own blog URL

== Installation ==

= Installation =

1. Upload the file `kn-public-chat.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Please change the settings and enjoy!

= Updating =

If you had a trouble in updating this plugin, try deactive old version and delete the folder `kn-public-chat` from the `/wp-content/plugins/` directory. Then, try download new version and install as usual.
